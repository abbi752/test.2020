# 简介

Code Template 1是Basic基础组合，主要包含下面的控件和模块：

- BottomNavigation  底部栏导航
- ViewPager         滑动页面切换
- BaseActivity      重构Base类
- Lambda插件：gradle-retrolambda
- RxJava, RxAndroid 并发和通讯
